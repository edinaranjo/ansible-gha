Servidor Web Automatizado con Ansible

Laboratorio de Infraestructura como Código

DevOps | Ansible | Apache | Linux

Descripción

Esta página fue desplegada automáticamente mediante Ansible en un entorno local. El objetivo del laboratorio es demostrar cómo la configuración de infraestructura puede definirse, ejecutarse y repetirse mediante código.

Componentes implementados

  • Ambiente virtual de Python.
  • Instalación de Ansible.
  • Inventario local.
  • Pruebas de conectividad.
  • Instalación y configuración de Apache.
  • Despliegue automatizado de contenido web.

Resultado

El servidor Apache se encuentra activo y entregando contenido personalizado generado como parte de una práctica profesional de DevOps.
